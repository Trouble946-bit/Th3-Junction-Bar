# Th3 Junction BaR

Static multi-page website for **Th3 Junction BaR**.

## Pages
- `index.html`: Home page
- `alcohols.html`: Alcohol categories hub page
- `beer-cider.html`: Dedicated beer and cider page
- `whiskey-bourbon.html`: Dedicated whiskey and bourbon page
- `vodka.html`: Dedicated vodka page
- `gin.html`: Dedicated gin page
- `rum.html`: Dedicated rum page
- `tequila.html`: Dedicated tequila page
- `wine.html`: Dedicated wine page
- `non-alcoholic.html`: Dedicated non-alcoholic page
- `contact.html`: Contact information and inquiry form

## Run locally
Open `index.html` in a browser.

## Notes
- This is a static front-end project (no back-end form handling configured).
